PASO 1: Hacer backup de archivo actual
────────────────────────────────────────
1. Cierra Excel (guarda todo)
2. Copia el archivo Excel a: C:\Users\Desktop\SISTEMA_BACKUP_2026.xlsm
3. Abre el archivo original nuevamente

PASO 2: Instalar Modulo 4 (5-10x más rápido)
────────────────────────────────────────────

3. Click derecho -> Ver código

6. Abre: Modulo4_SeleccionFilas_OPTIMIZADO_v31.txt

8. Vuelve a Excel y Ctrl+V (pega)

──────────────────────────────────────────────

VII. SOPORTE Y TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════════

PROBLEMA: "Constant is not defined"
SOLUCIÓN:
   1. Verifica: Option Explicit al inicio
   2. Re-pega COMPLETO (desde Option Explicit hasta fin)

PROBLEMA: "Type Mismatch in ObtenerDatosAsignacion"
SOLUCIÓN:
   1. Abre GUIA_COMPARATIVA_v30_vs_v31.txt
   2. Busca: Acceso a Array
   3. Verifica: datosArray(1, i) (ambos índices separados)

PROBLEMA: "Still slow after update"
SOLUCIÓN:
   1. Cierra Excel completamente
   2. Abre nuevamente
   3. Ejecuta: Alt+F8 -> CargarCacheEnMemoria
   4. Intenta nuevamente

PROBLEMA: "Compiler error: Expected Sub, Function, or Property"
SOLUCIÓN:
   1. Olvidaste pegar Option Explicit
   2. Abre archivo .txt
   3. Verifica que comience con Option Explicit
   4. Re-pega completo
